A full-stack Instagram Clone built with MERN stack.

## Features
- **Authentication**: JWT-based Login/Signup.
- **Feed**: Infinite scroll feed with "Stories" bar.
- **Posts**: Upload photos with captions.
- **Engagement**: Like and Comment on posts.
- **Profile**: Grid view of user posts.

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS v4.
- **Backend**: Node.js, Express, MongoDB.

## Setup
1. **Backend**:
   ```bash
   cd backend
   # Add .env with MongoDB URI & Cloudinary Credentials
   npm install
   npm run dev
   ```
2. **Frontend**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
